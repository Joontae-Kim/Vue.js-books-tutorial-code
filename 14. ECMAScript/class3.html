14.4 모듈

  • 모듈은 JS에서 가장 향상된 기능이다.
  • ES6는 여러 파일에서 모듈을 내보내고(export), 가져올(import)수 있는 기능을 지원.
  • 예제
    1. 변수 출력 예제
      module.js 
          export name = 'Alex'
      main.js 
          import { name } from './module';
    console.log('Hello ', name) // "Hello Alex" 출력
    2. 변수 및 함수 출력 예제
      module.js 
          export name = 'Alex'
    export function getAge(){
      return 22;
    }
      main.js 
          import { name, getAge } from './module';
    console.log(name, ' is ', getAge()) // "Alex is 22" 출력
    3. 객체 내부에서 내보내는 예제
      module.js 
          var name = 'Alex'
    function getAge() {
      return 22;
    }
    export defualt { name, getAge}
      main.js 
          import person from './module';
    console.log(person.name, ' is ', person.getAge()) // "Alex is 22" 출력
