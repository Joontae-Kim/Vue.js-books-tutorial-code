  • ES6를 이용하면 기본 매개벼수 값을 정의할 수 있음.
  • function divide(x, y = 2) {
  return x/y;
}

// 다음 코드와 동일
function divide(x, y){
  y = y == undefined ? 2 : y;
  return x/y;
}

14.7 템플릿 리터럴

  • 템플릿 리터럴은
  • 표현식을 포함할 수 있는 문자열 리터럴로써, 여러 줄 문자열과 보간 기능을 지원함.
  • 백틱(``)으로 감싸며, 백틱 안에서는 `$(표현식)`으로 사용할 수 있음. 이때 표현식은 함수나 변수 또는 실제 표현식이 될 수 있음.
    1. 변수 사용 
      let name = 'Alex'
console.log(`Hello ${name}`)
    2. 함수 사용 
      add = (a, b) => a + b;
let [a, b] = [10, 2]

console.log(`if you have ${a} eggs and you buy ${b} more yout'll have ${add(a,b)} eggs!`)
    3. 표현식 사용 
      let [a, b]= [10, 2]
console.log(`if you have ${a} eggs and you buy ${b} more yout'll have ${ a + b } eggs!`)
