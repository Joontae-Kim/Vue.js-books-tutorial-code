14.2 변수선언

  1. let 선언
    • 변수를 let으로 선언할 경우 변수의 유효 범위가 현재 코드 블록으로 제한됨.
    • let으로 선언한 변수는 감싸진 {}블록의 위호 호이스팅되지 않기 떄문에 블록 전체에서 변수를 사용할 수 있도록
    • let 선언을 블록에서 맨 위에 두는 것이 좋다.
    • if 안의 let 
        let age = 22
  if(age >= 18){
    let adult = true
    console.log(adult) // true 출력
  }
  console.log(adult) // adult에 접근 불가
  console.log(age) // age 출력
    • 맨 위에 선언한 let 
        let age = 22
  let adult
  if(age >= 18){
    adult = true
    console.log(adult) // true 출력
  }
  console.log(adult) // adult에 접근 가능
  console.log(age) // age 출력
  2. const 선언
    • let 선언과 마찬가지로 블록 단위로 선언됨.
    • 차이점은, const를 이용해 변수를 선언하면 해당 변수는 상수가 됨. 즉, 값을 더 이상 변경할 수 없음.
