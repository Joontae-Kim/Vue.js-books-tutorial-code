디렉티브

1. v-show 디렉티브

  • v-show가 지정된 엘리먼트는 항상 렌더링되어 DOM에 유지됨.
  • v-show는 단순히 엘리먼트의 display CSS 속성을 토글하는 역할을 함.

예제 1)

  • v-model을 이용해 textarea를 message 변수와 바인딩해서 메시지를 표시
{{ $data }}

예제 2)

  • v-model을 이용해 textarea를 message 변수와 바인딩해서 메시지를 표시
  • v-show를 이용해 button를 message 변수와 바인딩해서 message 변수의 상태에 따라 button을 토글하는 예제
{{ $data }}

예제 3)

  • v-model을 이용해 textarea를 message 변수와 바인딩해서 메시지를 표시
  • v-show를 이용해 button를 message 변수와 바인딩해서 message 변수의 상태에 따라 button을 토글
  • @click 이벤트에 alert 메소드를 바인딩해서 message를 alert창의 메시지로 활용하는 예제
