상호작용

이벤트 처리

  • HTML 페이지에서 Vue.js를 사용하면 Vue.js가 이러한 이벤트에 반응합니다.
  • 이벤트는 기본적인 사용자 상호작용에서 렌더링 모델에서 일어나는 일에 이르기까지 모든 것을 나타낼 수 있음
  • HTML 이벤트의 몇 가지 예
    • 웹 페이지가 완전히 로드됨
    • 입력 폼이 변경됨
    • 버튼이 클릭됨
    • 폼이 제출됨
  • 이벤트 처리의 핵심
    • 이벤트가 발생할 때마다 무언가를 할 수 있다는 것
    • Vue.js에서 DOM 이벤트를 청취하려면 v-on 디렉티브를 사용하면 됨.
  • v-on 디렉티브는
    1. 이벤트 리스너를 엘리먼트에 붙입니다.
    2. 이벤트 유형은 인자로 표시
    3. 예) v-on:keyup : keyup이벤트를 청취
