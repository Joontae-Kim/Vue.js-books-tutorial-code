상호작용

키 한정자 또는 키 수식어

  • 입력 엘리먼트 중 하나에 포커스를 주고 엔터키를 누르면 calculate 메서드가 호출됨
  • 폼에 버튼이 없거나 버튼이 전혀 없는 경우에는 키보드 이벤트를 대신 청취할 수도 있음
  • 키보드 이벤트를 청취할 때는 키 코드를 확인해야 할때가 많음..
  • 키 수식어 별칭 목록
    • .enter
    • .tab
    • .delete (“Delete” 와 “Backspace” 키 모두를 캡처합니다)
    • .esc
    • .space
    • .up
    • .down
    • .left
    • .right

예제

  • input 엘리먼트에 @keyup.enter="calculate" 키 한정자를 바인딩에 calculate 버튼을 누르거나 클릭하지 않아도
  • input 엘리먼트의 값을 변경한 후, Enter 키를 누르면 자동으로 계산값이 변경됨.
