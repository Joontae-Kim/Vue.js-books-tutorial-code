컴포넌트

7.1 정의

  • 기본 HTML 엘리먼트를 확장해 재사용 가능한 코드를 캡슐화 하는데 이바지함.
  • Vue.js의 컴파일러가 특정 동작을 추가할 사용자 정의 엘리먼트에 해당.
  • 어떤 경우에는 특별한 is 속성으로 확장된 기본 HTML 엘리먼트로 나타나기도 함.
  • 새로운 기능을 위해 HTML을 확장하는 영리하고 강력한 방법

7.2 컴포넌트 사용하기

  • 컴포넌트 등록 방법
    • Vue.component 메서드를 사용
    • 태그(tag)와 생성자(constructor)를 전달하는 방식으로 아래와 같이 생각하면 됨.
      • 태그(tag) = 컴포넌트의 이름
      • 생성자(constructor) = 생성자를 옵션
예제
  • 컴포넌트 이름(태그) = story
  • 프로퍼티 = story
  • story 컴포넌트 등록 
    Vue.component('story', {
  template: '<\h1>My horse is amazing!<\/h1>'
})
  • 그리고 HTML 안에 <\story><\/story>라는 사용자 정의 엘리먼트를 추가
