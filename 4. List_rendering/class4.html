v-for를 이용한 객체 프로퍼티 순회

  1. 배열의 index를 표시하듯 객체를 순회할 때도 똑같이 할 수 있음.
    1. index 외에도 각 유효범위 내에서 key라는 특별한 프로퍼티에 접근가능

예제1

  • v-for 디렉티브에서 value를 인자로 지정하면 각 프로퍼티인 plot, writer, upvotes의 값을 li 태그의 인자로 받는다
  • {{ value }}

예제2

  • key와 index를 각각 두 번째와 세 번째 인자로 지정할 수 있음.
  • {{ index }} : {{ key }} : {{ value }}
